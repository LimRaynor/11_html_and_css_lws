CSS 기본 선택자

선택자(Selector) 특정한 HTML 요소를 선택할 때 사용하는 기호

tag selector

   - HTML문서 내에 지정된 태그를 갖는 모든 요소를 선택하는 선택자

p{color : red;} 작성하기

ID 선택자 (#) 

    - HTML문서 내에 해당하는 아이디 속성을 가진 태그만 선택할 때 사용하는 선택자
       - HTML문서 내에 중복되는 아이디를 가지는 요소는 존재해선 안된다!

  
    [작성법]
    #id속성값 { css code; }

class selector(.)

    HTML 문서 내에 여러개의 요소를 동일한 클래슴명을 부여하고
    선택할 때 사용한다. (여러요소 선택시 주로 사용)
    [작성법]
    .class속성명{ css code; }
